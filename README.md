# rvemu-d

A Risc-V Emulator in D

Based on <https://book.rvemu.app/> and <https://siriusdemon.github.io/Rare/>

## Progress

- [x] Adder
- [x] Memory and System Bus
- [x] Control and Status Registers
- [x] Privilege Modes
- [x] Exceptions
- [x] PLIC & CLINT
- [x] UART
- [x] Interrupts
- [x] Virtio
- [x] Virtual Memory System and Page Table
- [x] Booting up xv6
